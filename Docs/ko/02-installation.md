# 2. 설치

## 요구 환경

- **.NET 10.0** 이상

## 설치 방법

### 1. GitHub Releases에서 받기

[Releases 페이지](https://github.com/bluekms/StaticDataPipeline/releases) 에서 CLI 도구 바이너리(zip) 를 받아 압축을 풉니다.

### 2. 소스 빌드

```bash
git clone https://github.com/bluekms/StaticDataPipeline.git
cd StaticDataPipeline
dotnet build -c Release
```
