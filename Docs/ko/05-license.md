# 5. 라이선스

StaticDataPipeline은 **MIT 라이선스** 로 배포됩니다.

전체 문구는 리포지토리 루트의 [`LICENSE`](../../LICENSE) 파일을 참고하세요.

## 요약

- 상용·비상용 구분 없이 자유롭게 사용·수정·재배포할 수 있습니다.
- 재배포 시 저작권 표시와 라이선스 전문을 포함해야 합니다.
- 무보증입니다. 사용으로 인한 피해에 대해 저작권자는 책임을 지지 않습니다.

## 기여

이슈 제보·기능 제안·PR 은 [GitHub Issues](https://github.com/bluekms/StaticDataPipeline/issues) 로 부탁드립니다.
